# Instructions
In order to run this you it is often best to run from the command line

Compile from the src dir:
javac -d ../bin exams/finalexam/OubliettesOfPeril.java

Execute from src dir:
java -cp ../bin exams.finalexam.OubliettesOfPeril

With args:
java -cp ../bin exams.finalexam.OubliettesOfPeril "Arguements"
